# Sistema de Avaliação de Empréstimos Bancários
---
Este projeto implementa um sistema em C para análise de solicitações de empréstimo com base em critérios financeiros. A lógica envolve o cálculo de endividamento, avaliação da capacidade de pagamento e verificação da estabilidade no emprego.
# Fluxo de Funcionamento

O sistema segue a seguinte lógica:

1. Recebimento das informações financeiras e pessoais relevantes.  
2. Cálculo dos indicadores utilizados para avaliação.  
3. Validação sequencial de cada critério estabelecido.  
4. Emissão da decisão final de aprovação ou recusa com base nas regras definidas.

##Essa estrutura organiza o processo, facilita futuras expansões e mantém a tomada de decisão precisa.
